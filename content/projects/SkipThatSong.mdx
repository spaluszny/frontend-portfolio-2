# Audio Engineering and Sound Production Portfolio
## By: Sarah Paluszny

Framer is a no-code website builder that has been increaing in popularity in the last couple of years.
I thought I'd check it out.

For this project, I created a fictional product called Skip-That-Song. Skip-That-Song is an app that allows
users to skip the current karaoke perfomance. It has featues such as democratic skipping, skip analytics, and
easy venue integration. 

I started this project by first designing the website in Figma

## My Review of Framer

Having web developer experience I found it easy to create a functional website. Having used flex box many times before
I found it easy when handling resize properties in framer. I could see how this might be difficult for a first time 
web user as Framer claiming the easy of drag and drop did not come very intuitively 

Framer made it easy to create a static website with interactivity. The mobile and desktop breakpoints made it
easy to handle resizing and I was impressed with easy it made a sliding nav bar for mobile. However, I did notice some 
bugs with the mobile nav bar. When you open it and click on. 

Since I used a preset nav bar offered in Framer, I couldn't find a way to stop this from happening. 

Although it isn't perfect yet I could see Framer taking over as the top the no-code website builder.
