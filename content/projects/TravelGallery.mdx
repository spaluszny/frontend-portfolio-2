<p className='drop-cap'>Recently I embarked on a journey to Nepal to do the Manaslu Trek. It was a wonderful adventure filled with angelic mountains, bewhiching waterfalls, magnificent wild life. As someone who travels a lot I'm always being asked by my family and friends to send pictures and I've always struggled picking the top ones to send to each individual. I decided to make a travel photo gallery to solve this issue.</p>
<br/>

### Overview
On the user frontend I wanted it to clean and easy to use. Photos broken out by continents and an image lightbox to see more information about each photo. Simple concept, but I knew if I didn't have a light cms integration for photo posting, I would never update the website. 
<br/>
**Goal: Create a photo gallery website with easy to use image upload**

<br/>
### The Problem
<br/>
**FEATURES**
* User dashboard to easily upload photos
* Photos displayed
* Photo lightbox
* Join the newletter
<br/>
For the design, I decided to go with a sidebar that contained all site navigation and on the left a masonry grid of images that are filtered by continent. 
<br/>
### Technical Approach
Since all of my travel photos already lived on google photos, I wanted to find a way to sync them to my website. When browsing my google photos from with developer tools, I found that with public google photos links you can scrap the html to find the image previews.  

I was then able to write a react component that took the public google photos url and return the associated images from that folder. I could easily add photos to my public photos album and it would instantly show up on my site. 
<br/>
### Challenges
Although the functionality worked great, I ran into a lot of flaws with this first iterations. The biggest one being the image quality. The photos in the public google photos url html only contained the preview images, causing them to show up grainy when blown up. Another flaw was I couldn't access any photo metadata. This was another main feature I wanted to implement, causing me to go to a new iteration.
