# Audio Engineering and Sound Production Portfolio
## By: Sarah Paluszny

During my time at IU, I spent countless hours in the recording studio. To document that experience, I created a web portfolio. I wanted to return to the basics of front-end development, so I built it using the three core languages: HTML, CSS, and JavaScript. It was a great way to apply my existing knowledge while also learning how to optimize web pages without relying on prebuilt libraries.
